## 食道
- GERD
	- 胸骨下灼熱 (Heart burn)
	- 充血 -> 腐蝕 -> 潰瘍
	- 併發 : [[新同學#^f8395d|Barett esophagus]]
- Esophageal atresia
	- 症狀 : 出生前羊水過多、出生後一吃就吐、反覆性肺炎、脹氣
	- [[食道閉鎖分型.png]]
	- 合併先天性異常 : VACTERL (Hemivertebra、Imperforate anus、Cardiac defect、Trachea-esophagus anormaly、Renal deformity、Limb deformity)
- Esophageal cancer
	- Barett -> Adeno、下1/3
	- Web -> Squamous、上1/3
	- Achalaisa -> Squamous、下1/3
	- 病理 : 90% Squamous、10% Adeno
- [[syndrome#^cc54d5|Mallory-Weiss syndrome]]
## 胃
- Pyloric stenosis
	- 先天性
		- 缺Auerbach plexus，胃竇環肌肥大
		- Projectile vomiting、無膽汁
		- 滿月時嘔吐、Palpable olive mass
		- 消化道攝影String sign
	- 後天性 (常見) : 潰瘍後纖維化
### 胃炎
- 急性胃炎 : NSAID、酒精、壓力 (休克、Cushing ulcer、Curling ulcer)
- 慢性胃炎

|  | Type A | Type B |
| ---- | ---- | ---- |
| 又名 | 萎縮性、<br>自體免疫性 | 發炎性、感染性 |
| 原因 | Parietal cell、 內在因子 Ab | H.pylori |
| 病理 | 胃黏膜萎縮、<br>G cell增生、<br>惡性貧血 | 侵犯**胃竇**黏膜**表面**、Hyperacidity、MALToma<br>Intestinal metaplasia |
- #h/blue ==比較== MALToma : 唾腺 (Sjogren)、甲狀腺 (Hashimoto)、胃 (H.pylori)
- H.pylori破壞D cell、分泌Urease破壞黏膜，造成95%十二指腸潰瘍，75%胃潰瘍

## 腸道
### Infective enteritis
- Toxin (黏膜完整)
	- Preformed : S.aureus、V.cholerae、C.perfringens
	- Toxigenic : ETEC
- Invasive
	- C.jejunui (**最常見細菌**)、Rotavirus (**最常見病毒**)、Shigella、EHEC、E.histolytica
	- TB : 腸結核、橫走性潰瘍 (迴腸)
	- S.typhi : 縱走性潰瘍 (迴腸)
		- 症狀 : 1w菌血症、2w侵犯巨噬細胞、3w腸道潰瘍
### IBD
- Idiopathic inflammatory bowel disease)
- 原因 : 基因、腸內微生物、上皮缺損、異常免疫
- 治療 : 5-ASA (Sulfasalazine) + Sulfapyridine

|  | Crohn's disease | Ulcerative colitis |
| ---- | ---- | ---- |
| 表徵 | 吸收不良 | 血便性腹瀉 |
| 部位 | **Ileocecal** valve附近 | 大腸 (**直腸往上**) |
| 分布 | skip lesions | continuous |
| 病灶 | 全層 (深而窄) | 黏膜層 (淺而廣) |
| 病理<br>特徵 | **Noncaseating granuloma** | Crypt abscess、**Pseudopolyp** |
| 黏膜<br>鏡檢 | 線性潰瘍 (縱裂)、<br>鵝卵石樣、腸道狹窄 | 瀰漫性充血、<br>融合型潰瘍 |
| 免疫 | Th1 | Th2、NKT cell |
- Cobble stone : 縱向和橫向潰瘍+黏膜水腫
- Pseudopolyp : 留下的正常黏膜像息肉
- 可能對腸內microbiota有強烈發炎反應
