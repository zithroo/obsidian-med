## 原發性
|        | Adeno          | Squamous                              | Small cell             | Large cell |
|--------|----------------|---------------------------------------|------------------------|------------|
| 發生率 | 40%            | 30%                                   | 20%                    | 10%        |
| 位置   | 周圍           | 中央                                  | 中央                   | 周圍       |
| 基因   | **EGFR、K-RAS**    | p53                                   | c-MYC、c-KIT、p53、RB  |            |
| 好發   | 不抽女         | 抽菸老男                              | 抽菸                   |            |
| 病理   |                | Keratin pearl、Intercellular bridging | 小藍圓、燕麥細胞       |            |
| 特色   | 生長慢、快轉移 | 生長快、慢轉移、**PTH**                        | 神經內分泌、最惡性 |            |
