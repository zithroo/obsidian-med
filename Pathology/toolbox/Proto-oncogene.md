	- EML4-**ALK** fusion protein : 肺腺癌

## Signal transduction
- Ras : G protein、有GTPase活性出現**點突變**
- ABL-BCR (9 : 22) : 費城染色體
- RTK (例如EGFR) -> 
	- SOS -> Ras -> Raf -> MEK -> ERK (**MAPK**)
		- SOS : Guanine Exchange Factor，把Ras上的GDP換成GTP活化
	- PI3K -> AKT -> mTOR
