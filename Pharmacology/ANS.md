## Ganglion blocker
- 機轉 : 作用於節後神經，**非專一影響主導支配的神經**
	- Sym : 血管平滑肌、汗腺
	- Para : 其他organ
### 藥物
1. Depolarizing blocker : Nicotine
	- 去極化 -> activation -> incativation
2. Competitive blocker : #h/red  ==踹妹、妹卡、嘿傻妹==
	- Trimethaphan : **惡性高血壓**
	- Mecamylamine : 降血壓
	- Hexamethonium
