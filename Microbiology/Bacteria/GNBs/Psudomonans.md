## P.aeruginosa
- 絕對需氧
- 色素 (墨綠色金屬)、葡萄味
- [[抑制蛋白質合成毒素]]
- **想到水**

|        | 位置         | 症狀                  |
|--------|--------------|-----------------------|
| 正常   | 肺           | Cystic Fibrosis人肺炎 |
|        | 眼           | 隱形眼鏡->角膜潰瘍    |
|        | 耳           | swimming ear (外耳)   |
| 伺機性 | 傷口、導尿管 |                       |
## P.pseudomallei
- 現名 : Burkholderia psudomallei
- **類鼻疽** (Meilioidosis) : 皮膚潰瘍、肺炎、敗血症
